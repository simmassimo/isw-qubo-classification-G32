The functions in `sklearn.metrics` provide all main metrics and structured reports; `classification_report` and `precision_recall_fscore_support` return precision, recall, f1 and support per class.

### Quick Python example
```python

import numpy as np

# y_true, y_pred, y_score are your arrays
accuracy = accuracy_score(y_true, y_pred)
prec, rec, f1, support = precision_recall_fscore_support(y_true, y_pred, labels=[0,1])

```

This flow derives exactly the fields in your JSON using standard functions.

### Practical notes and recommendations
- **Support** is simply the count of examples per class (using `support` from `precision_recall_fscore_support` avoids recounting).  
- For **roc_auc** you need a vector of scores/probabilities (`predict_proba` or `decision_function`), not the predicted labels.  
- For imbalanced datasets consider also `average_precision_score` or Precision‑Recall curves as complements to ROC.
