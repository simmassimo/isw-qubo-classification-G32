## Testing
From the project directory use the following command:
- `pytest tests/test_01.py`
or similar ...

Find the input files in `tests/data` with names like:
- `input01.csv`

Find the output files in `tests/data` with names like
- `normalize01.csv`
- `report01.json`

Also in `tests/data` are files with names like
- `test_data01.json`
these are only used for debugging purposes

To date only `preprocessing.py` can be tested
